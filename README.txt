PAGINA DE PERDON - LISTA PARA GITHUB PAGES

Archivos:
- index.html
- style.css
- script.js
- foto.jpg

QUE HACER:
1. Reemplaza foto.jpg por tu imagen.
2. Si quieres cambiar frases, edita el arreglo "words" en script.js.
3. Para cambiar el mensaje principal, edita index.html.
4. Sube todos los archivos a un repositorio en GitHub.
5. En GitHub, entra a Settings > Pages y publica desde la rama principal.

EFECTOS:
- La imagen gira 180 grados al pasar el mouse o tocar el boton.
- Hay muchas palabras flotando, estilo parecido a tu ejemplo, pero pidiendo perdon.
